## Desafio proposto no desafio do módulo 2 do curso de react do IGTI.
### Desafio: Criar uma interface para visualização dos dados de uma eleição ficticia

Devem ser apresentados os dados:
 * Porcentagem
 * Quantidade dos votos
 * Quem foi eleito
 * Total de eleitores
 * Abstenções
 * Comparecimentos
